# DEWA III EHS Command Center

Interactive front-end dashboard for the Restoration of DEWA Phase III 800 MW PV Solar Power Project.

## Included modules

- Executive dashboard
- Safety observations
- Incidents and near misses
- Permit to Work
- Inspections and audits
- Risk register
- Plant and equipment
- Environmental management
- Welfare and heat stress
- Emergency resources
- Training and competency
- Contractor performance
- Reports and analytics

This repository currently contains a static prototype prepared for later integration with dewa3hse.com and a WordPress-based EHS management system.
